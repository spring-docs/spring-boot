== Installing the CLI
The Spring Boot CLI (Command-Line Interface) can be installed manually by using SDKMAN! (the SDK Manager) or by using Homebrew or MacPorts if you are an OSX user.
See _xref:getting-started:getting-started-installing-spring-boot:getting-started-installing-the-cli.adoc#getting-started-installing-the-cli[getting-started-installing-the-cli]_ in the "`Getting started`" section for comprehensive installation instructions.
